عن فلورابيت

منصة متكاملة للعناية بالنباتات المنزلية.

الفكرة

نظام ذكي يجمع تطبيق جوال لأصحاب النباتات مع لوحة ويب للإدارة، وخادم API موحّد، وقاعدة بيانات SQLite، ونموذج تعلّم آلي للتعرف على النبات من الصورة، مع كتالوج عربي ومواعيد ري وتسميد وتنبيهات وسجل عناية.

المكوّنات

  • تطبيق Flutter (Android) للمستخدم النهائي
  • لوحة ويب Flask للمسؤولين
  • REST API مشترك مع التطبيق
  • تعرف على الصورة (RandomForest مدرب)
